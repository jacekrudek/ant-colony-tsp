# ant-colony-tsp

**Visualisation of the Ant Colony Optimazation algorithm solving the Travelling Salesman Problem, built with Python & Pygame.**

# Development

## Prerequisites

This project requires [python](https://www.python.org/downloads/) version 3.12 or newer.

Make sure **python** is installed correctly.

```
py --version
```

Additionally, the following dependencies are required: 
- pygame-ce
- pygame_gui
- python-i18n

## How to launch app
```bash
pip install -r requirements.txt
py main.py
```

The solution will be working as a desktop app.
